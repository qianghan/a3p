## WhatsApp

Prefer WhatsApp to Telegram? AgentBook has one shared number you message to link your account — after that it works exactly like chatting with the agent anywhere else.

### Connect WhatsApp

1. Go to **Settings → Chatbots** and open the **WhatsApp** card. It shows the AgentBook WhatsApp number and a one-time code just for you.

   ![Settings, Chatbots tab, WhatsApp card not yet connected, showing the AgentBook WhatsApp number and a one-time link code](/docs/whatsapp/settings-chatbots-whatsapp-not-connected.svg)

2. Open WhatsApp, message that number, and send your code. You'll get a confirmation reply right away — that number is now tied to your AgentBook account.

   ![WhatsApp conversation: sending the link code to the AgentBook number and receiving a connected confirmation](/docs/whatsapp/whatsapp-link-chat.svg)

3. The Chatbots card now shows your linked number, and you're ready to go.

   ![WhatsApp card after connecting, showing the linked phone number and a disconnect option](/docs/whatsapp/settings-chatbots-whatsapp-connected.svg)

From WhatsApp you can log an expense — *"log $12 parking"* — or ask a question about your books, same as Telegram. WhatsApp is currently text-only; for receipt photos and inline buttons, use Telegram or the web app.

> **Tip:** Disconnecting just unlinks your phone number — your code stays the same, so you can relink anytime without going through setup again.
