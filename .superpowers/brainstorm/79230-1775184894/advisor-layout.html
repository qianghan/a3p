A. Floating Chat Bubble

WhatsApp-style chat bubble on the expense page. Click to open a chat panel. Non-intrusive, available everywhere.

  • Familiar pattern (Intercom, Drift)
  • Available on every page
  • Doesn't take space until opened
  • Chat format limits charts/visuals
  • Small window for rich content

B. Side Panel (Split View)

Persistent panel on the right side of the expense page. Shows proactive insights, charts, and has a chat input. Like GitHub Copilot sidebar.

  • Always visible — proactive insights surface naturally
  • Enough space for charts + rich content
  • Context-aware (sees what you're browsing)
  • Takes screen real estate
  • Only on expense page

C. Inline Advisor (Recommended)

AI insights appear as cards within the expense page flow. Proactive alerts at top, expandable charts, and a command bar for questions — all integrated into the page, not separate.

  • Feels native — not bolted on
  • Proactive insights are the first thing you see
  • Charts expand inline with full width
  • Works perfectly on mobile
  • Same pattern can work in Telegram (cards → messages)
  • More complex to build
  • Need good visual hierarchy to not overwhelm
